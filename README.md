## 🎮 游戏机制

### 蛇的控制
```
方向键控制:
  ↑ / W - 向上移动
  ↓ / S - 向下移动
  ← / A - 向左移动
  → / D - 向右移动

功能键:
  空格键 - 开始/暂停/继续
```

### 难度设置
| 难度 | 速度 | 说明 |
|------|------|------|
| 简单 | 200ms/帧 | 适合新手,移动速度较慢 |
| 中等 | 150ms/帧 | 标准速度,平衡挑战性 |
| 困难 | 100ms/帧 | 高速移动,考验反应能力 |

### 计分规则
- 每吃一个食物: **+10分**
- 游戏结束时自动保存最高分
- 最高分永久保存在浏览器本地

### 游戏状态
1. **准备状态** (ready): 初始状态,等待开始
2. **游戏中** (playing): 正在进行游戏
3. **暂停状态** (paused): 游戏暂停
4. **游戏结束** (gameover): 碰撞后结束

## 🔧 技术实现细节

### Canvas绘制
```javascript
// 画布规格
尺寸: 400x400 像素
网格: 20x20 格子
单格: 20x20 像素

// 绘制顺序
1. 清空画布(深色背景)
2. 绘制网格线(可选)
3. 绘制食物(红色方块)
4. 绘制蛇身(绿色方块)
5. 绘制蛇头(深绿方块)
```

### 数据结构
```javascript
// 蛇的数据结构
snake = [
  {x: 10, y: 10},  // 头部
  {x: 9, y: 10},   // 身体
  {x: 8, y: 10}    // 尾部
]

// 食物数据结构
food = {x: 15, y: 12}

// 游戏状态
gameState = 'ready' | 'playing' | 'paused' | 'gameover'
```

### 移动算法
```javascript
1. 根据方向计算新头部位置
2. 将新头部添加到数组前端(unshift)
3. 检查是否吃到食物:
   - 是: 保留尾部(蛇变长)
   - 否: 移除尾部(pop,保持长度)
```

### 碰撞检测
```javascript
撞墙: head.x < 0 || head.x >= 20 || head.y < 0 || head.y >= 20
撞己: snake.some(segment => segment.x === head.x && segment.y === head.y)
吃物: head.x === food.x && head.y === food.y
```
